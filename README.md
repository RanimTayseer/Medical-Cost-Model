# 🌸 FlowerCatch Game

A 2D interactive arcade-style game built with **Python** and **Tkinter**, where players catch falling flowers within a limited time. The game combines simple mechanics with visual effects, progressive difficulty, and a scoring system to create an engaging mini-game experience.


## 🎮 Game Overview

In **FlowerCatch Game**, flowers fall from the top of the screen, and the player controls a basket to catch them. Each collected flower increases the score and is visually added to a decorative vase. The game becomes progressively harder as the level increases.


## ✨ Features

- ⏱️ **60-second timed gameplay**
- 🎯 **Score system**
- 📈 **Dynamic difficulty levels**
- 💥 **Particle effects on collection**
- 🔊 **Sound feedback (system beep)**
- 🏆 **High score saving system**
- 🌸 **Multiple flower types & sizes**
- 🏺 **Visual vase filling effect**


## 🧠 Game Mechanics

- Flowers spawn randomly at the top of the screen.
- Player moves left/right using arrow keys.
- Catching flowers increases score.
- Difficulty increases every few points.
- Game ends when time reaches zero.


## 🕹️ Controls

| Key | Action |
|-----|--------|
| ⬅️ Left Arrow | Move left |
| ➡️ Right Arrow | Move right |
